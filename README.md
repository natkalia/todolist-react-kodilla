# To-Do list App with React :clipboard:

To-Do List app created as exercise for Web Developer Bootcamp in Kodilla (Module 12.2).

## Technologies/tools

Project is created with:
* HTML
* Sass for CSS
* Javascript (ES6)
* React
* Redux
* deployment on Heroku with http-server package
* Webpack
